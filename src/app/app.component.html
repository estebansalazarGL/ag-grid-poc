AG Grid PoC

What's implemented?

  1. Sorting
  2. Basic filters (text, date, numbers)
  3. Column ordering
  4. Show/Hide columns
  5. Pagination
  6. Resizing
  7. Style grid
  8. Export to CSV
  9. Insert custom components in cell

Negative points

  • Row drag doesn't work with pagination
  • You must create the date filter
  • Row grouping is enterprise
  • You can export to CVS and Excel but Excel is enterprise
